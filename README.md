# recent-agents

A [zenbu](https://github.com/zenbu-labs/zenbu) plugin that adds a `Cmd+P` command palette to the chat view for switching between recently-used chats

## What it does

- Press `Cmd+P` inside a chat view to open the palette.
- Lists chats in MRU order (most recently switched-to on top).
- The top two items are swapped on open, so `Cmd+P Enter` toggles between the last two chats
